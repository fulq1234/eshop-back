基本数据类型

byte格式:${byteval!0}
int格式:${intval!0}
long格式:${longval!0}
boolean格式:${booleanval!0}
date格式:${dateval?string('yyyy-mm-dd HH:mm:ss')}
date,mysql 格式:${dateval2}
不存在的值:${nullval!'没有'}

遍历List


<#list listval as temp> ${temp},

遍历map


<#list mapval?keys as temp> ${temp}:${mapval[temp]}

if语法

<#if ifval == 1> 是1 <#elseif ifval ==2> 是2 <#else> 其它

switch语法,switch,case,break,default

<#switch ifval> <#case 1> 是1啊 <#break> <#case 2> 是2啊 <#break> <#default> 其它

字符串操作

<#assign a ="hello"> <#assign b ="world">
  • 连接
    • ${a+b}
  • 截取
    • ${(a + b)?substring(0,2)}
  • 长度
    • ${(a + b)?length}
  • 大写
    • ${(a + b)?upper_case}
  • 小写
    • ${(a + b)?lower_case}
  • index_of
    • ${(a + b)?index_of("w")}
