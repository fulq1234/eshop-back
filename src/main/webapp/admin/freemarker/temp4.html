宏macro,nested,return指令

  • 例子1：无参数的macro
    • <#macro test> 我是无参数的macro <@test/>
  • 例子2：有参数的macro
    • <#macro test2 param1 param2> 我是有参数的macro${param1}.${param2} <@test2 param1="java" param2="python"/>
  • 例子3：有默认参数的macro
    • <#macro test3 param1 param2="python"> 我是有默认参数的macro${param1},${param2} <@test3 param1 param2/>

    2.nested

    <#macro test param1="java"> ${param1}
    <#nested param1,"我的nested参数">
  • 调用
    • <#--param1="java"是参数,分号后面的是嵌套的内容,两个参数以分号分割--> <@test param1="java";loopVar1,loopVar2> hello ${loopVar1},${loopVar2} <@test param1="python";loopVar1> hello ${loopVar1}

    3,函数

    <#function doAdd param1 param2> <#return param1 + param2>
  • 调用
    • 你好，我是调用${doAdd(100,100)}
